Ecosystem

Mint Token

Receive BNB as incentive on minting $RXGS token         

Dapp allows its users to receive native token $RXGS in the meta mask wallet on minting using BNB. Once the hatching process is complete, the incentive in base currency BNB will get generated. Dapp has an option to redeem the incentives into the meta mask wallet.

Coin Box

Incentive Lottery - Receive BNB as incentive on booking ticket & also chance to win prize

Dapp allows its users to book lottery tickets with BNB currency. Receive the native token $RXGS as payment receipt in the meta mask. A user is eligible for the base currency BNB as an incentive from the EVM network. Also an opportunity to win a prize on the user luck.
